Privacy Policy for Timezone Helper

Data Collection and Usage

Timezone Helper is committed to protecting your privacy. This extension:

  • Only stores timezone preferences locally on your device using Chrome's storage API
  • Does not collect any personal information
  • Does not share any data with third parties
  • Does not track user behavior

Data Storage

  • All data is stored locally on your device
  • Data is only used to display timezone information in Google Calendar
  • No data is transmitted to external servers

Permissions Used

  • storage: Used to save your timezone preferences
  • activeTab: Required to interact with Google Calendar
